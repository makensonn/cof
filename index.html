Programming Is Fun

Why Program...

Programming let's you be creative. It let's you see how things work. If you ever thought of making an app, website, or developing software, then you should make that idea come to life and start programming.

What Programming Language...

There are many programming languages that you can learn, some of the most popular ones are JavaScript, C++, Java, and Python.

Get started...

Get started with W3Schools. W3Schools is a free educational website for learning to code.

Tools...

Visit Replit to begin your software projects. Replit is a simple yet powerful online IDE, Editor, Compiler, Interpreter, and REPL.

Careers

Why not get paid for something you'll love doing, there are a variety of jobs in the market, by learning to program you can become a Computer Programmer, Software Developer, Application Tester, Web Developer and more.
